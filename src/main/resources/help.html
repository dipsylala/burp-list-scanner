A tool to bulk-load sites into the Site Map.
Format: Sites should include the protocol, the site, and optionally a path. If the port isn't specified, the default is used for the protocol.

  • Paste - Takes values on the clipboard and adds them to the list.
  • Load... - Loads values from a file. Text file with one URL per line.
  • Remove - Remove the selected values from the list.
  • Clear - Empties the list.
  • Add - Allows the user to add a single line to the list.
  • Import to Site List - Takes the values in the site list, visits the site, and adds the request/response to the site list.
  • Clear (log) - Clears the log screen.

Options

  • Spider URLs - when the site is added to the site list, it's automatically added to the spider list too. Automatically adds the site to the scope to ensure the site can be spidered.
  • Follow Redirections - Anything in the 3xx Response range is analysed, and redirected according to the Location header. Redirection loops are detected.
  • Add to Scope - The URL is automatically added to the scope.

Notes

  • Timeouts - These are at the discretion of the project settings. If you want to change them, go to Project Options/Connections/Timeouts.
